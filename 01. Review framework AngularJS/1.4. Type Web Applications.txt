Типы веб-приложений:

-------------------------------------------------------------------------------------------------------------

Round-Trip Application

Классические веб приложения. Браузер запрашивает первую HTML страницу с сервера.
Последующие действия пользователя приводяк к загрузке полностью нового документа с сервера.

Недостатки:
Ожидание загрузки нового документа
Сложно контролировать состояние приложения
Больший объем данных передается по сети

-------------------------------------------------------------------------------------------------------------

Single Page Application

Браузер запрашивает первую HTML станицу, после чего общение с сервером происходит через AJAX запросы.
